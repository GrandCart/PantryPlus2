## Installation

1. Clone the repository
```bash
git clone https://github.com/GrandCart/PantryPlus2.git
```

2. Install dependencies via Swift Package Manager
```bash
cd PantryPlus2
swift package resolve
```

3. Open the project in Xcode
```bash
xed .
```

4. Build and run the project
